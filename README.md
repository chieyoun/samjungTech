# 📊프로젝트 개요

## 목표

⭐ 실제 졸업 앨범을 잃어버리거나 훼손되어도 볼 수 있게 온라인으로 졸업 앨범을 구현
<br>
⭐ 사용자가 계속해서 연락처를 수정할 수 있게 하여 동창과의 연결을 끊기지 않게 함

## 문제 정의

<aside>
💬 학교 동창이어도 졸업 앨범에 적힌 연락처가 바뀌면 연락을 하기 힘들어짐.

</aside>
